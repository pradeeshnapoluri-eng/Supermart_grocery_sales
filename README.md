# Supermart Grocery Sales Analysis

## Project Overview

This project analyzes sales data from a Supermart grocery store to understand customer purchasing behavior, product performance, and overall sales trends. The analysis helps identify insights that can improve business decisions and sales strategies.

## Objectives

* Analyze grocery store sales data
* Identify best-selling product categories
* Understand customer purchasing patterns
* Visualize sales trends and performance

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

## Dataset

The dataset includes grocery store transaction details such as:

* Order ID
* Product Category
* Sales Amount
* Profit
* Quantity Sold
* Customer Details
* Region and City

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## Key Analysis Performed

* Sales distribution across product categories
* Regional sales analysis
* Profit analysis by product type
* Monthly and yearly sales trends

## Results and Insights

The analysis highlights the most profitable product categories, regions with the highest sales, and trends in customer purchasing behavior.
